The tokenizer divides the Code into differnet Tokens

### Token List

| Token       | Description                                       |
| ----------- | ------------------------------------------------- |
| Keyword     | Any keyword like (if, var, out)                   |
| Literal     | Anything that can't be assigned to something else |
| Operator    | Examples: + - \|\| == >                           |
| Punktuation | only `:`                                          |
| Comment     | Comments                                          |
| Bracket     | () \[] {}                                         |
